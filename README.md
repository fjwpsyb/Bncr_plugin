# Bncr_plugin 简介

自用的无界插件。

# 安装与使用

在红灯区同级目录创建一个文件夹，名字随意 如：自用插件，把当前仓库所有文件粘贴到自用插件下

# 注意事项

<b>第一次使用请先执行 Doraemon_一键安装所需依赖.js 中 Doraemon_一键安装所需依赖 命令，安装所需依赖</b>
